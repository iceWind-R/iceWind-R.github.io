Docker安装Redis

使用命令docker pull redis 下载镜像

运行redis容器：docker run -itd –name redis01 -p 6379:6379 redis

使用命令行交互模式进入容器：docker exec -it redis01 /bin/bash

然后用redis-cli连接容器的redis进行测试
